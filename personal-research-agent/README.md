# Personal Research Agent - 个人研究员智能体

## 项目概述

Personal Research Agent 是一个基于 AI 智能体的研究辅助工具，能够：
- 理解用户的研究目标和需求
- 自动制定研究计划
- 调用多种工具检索资料
- 生成结构化的研究报告

### 目标用户
- 需要快速了解新领域的研究人员
- 进行文献综述的学生
- 需要资料整理的开发者

### 核心特性
- 🤖 智能体自主规划与执行
- 🔍 网络搜索与本地知识库检索
- 📊 自动摘要与报告生成
- 🛡️ 安全护栏与敏感信息过滤
- 💰 词元成本控制与预算管理

## AI Agent Playbook

### 任务目标
接收用户的研究目标，自动完成资料检索、信息提取和报告生成。

### 执行流程
1. **输入验证**: 安全检查用户输入
2. **计划生成**: 智能体制定3-5步研究计划
3. **步骤执行**: 依次执行搜索、检索、摘要等操作
4. **预算检查**: 每步检查词元消耗
5. **报告生成**: 汇总结果生成结构化报告

### 工具/API使用方式
- `web_search`: 模拟网络搜索（可替换为SerpAPI）
- `knowledge_base_search`: 本地知识库检索
- `research_summarizer`: AI驱动的摘要生成

### 关键决策逻辑
- 根据步骤描述自动选择合适工具
- 优先使用本地知识库以节省成本
- 超出预算时立即停止

### 异常处理策略
- 输入验证失败 → 返回错误信息
- 工具调用失败 → 记录错误，继续下一步
- 预算超限 → 停止执行，生成部分报告
